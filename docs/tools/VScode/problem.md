# VScode问题

### VSCode自动补全的函数不带括号,在setting.json中添加

```js
"typescript.suggest.completeFunctionCalls": true,
"javascript.suggest.completeFunctionCalls": true,
```

### vscode关闭声音提示

在搜索栏中输入 "editor.accessibilitySupport" 来查找相关设置，设置off。
